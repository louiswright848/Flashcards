Discrete Maps

Fixed points of Xₙ₊₁ = rXₙ / (1 + Xₙ²)
X = 0
X = √(r − 1) for r > 1
Derivative of the map
f'(x) = r(1 − x²) / (1 + x²)²
Stability: |f'(X*)| < 1

Predator–Prey

Nullclines of predator–prey system
x-nullclines: x = 0, y = (1 − x)/(1 + λ)
y-nullclines: y = 0, x = 1/λ
Equilibria of predator–prey
(0,0), (1,0), (1/λ, (1 − 1/λ)/(1 + λ))

Hardy–Weinberg

Genotype frequencies
AA = p², AB = 2pq, BB = q²
Fraction with at least one A allele
1 − q²

Evolutionary Game Theory

Replicator equation
ẋ = x(u_C − ū), where ū = xu_C + (1−x)u_D
ESS condition
u(s,s) > u(t,s) for all t ≠ s
